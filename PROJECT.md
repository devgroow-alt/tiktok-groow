# TikTok GROOW - Project Overview

Static Next.js clone of the tiktok.groow.sk Leadpages landing page.

## Tech Stack
- Next.js 16 (static export)
- TypeScript
- Tailwind CSS 4
- Google Fonts: Big Shoulders, Questrial

## Structure
- 13 section components in `src/components/`
- Single page app at `src/app/page.tsx`
- Static export to `out/` directory
- All images in `public/images/`

## Deployment
- GitHub: devgroow-alt/tiktok-groow
- Vercel: static deployment
- Target domain: tiktok.groow.sk
